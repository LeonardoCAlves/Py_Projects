<h3>📈Previsão de Preços do Ouro com Regressão Linear</h3>Este projeto utiliza a biblioteca scikit-learn para treinar um modelo de regressão linear e prever valores futuros com base em dados históricos de preços do ouro.

<h4>📝 Pipeline</h4>
<ul>
    <li><strong>Preparação dos Dados</strong>: Carregamento e pré-processamento dos dados.</li>
    <li><strong>Treinamento do Modelo</strong>: Treinamento de um modelo de Regressão Linear com os dados preparados.</li>
    <li><strong>Previsão</strong>: Previsão de preços futuros com base no modelo treinado.</li>
</ul>

<h4>🚀 Como Rodar o Projeto</h4>
1. Clone este Repositório
<pre><code>git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio</code></pre>
2. Instale as Dependências
<pre><code>pip install -r requirements.txt</code></pre>
3. Execute o Script
<pre><code>python main.py</code></pre>
